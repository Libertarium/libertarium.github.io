План реализации ЗАО "БСК" п. 23 лицензии Минсвязи N 5769 по обеспечению проведения ОРМ

29.01.1999
29 января 1999 года. "99/022

В соответствии с Конституцией Российской Федерации, с законами Российской Федерации "Об оперативно-розыскной деятельности", "О связи", "Об информации, информатизации и защите информации" (ст. 11), с Гражданским кодексом РФ (ч. 1. ст. 139), Уголовно-процессуальным кодексом РФ (ст.ст.12, 174), Уголовным кодексом РФ ст.ст. 137, 138 ЗАО "Байярд-Славия Коммуникэйшнс" предлагает следующий

ПРОЕКТ ПЛАНА

реализации ЗАО "Байярд-Славия Коммуникэйшс", предоставляющим услуги телематических служб на территории Волгоградской области, пункта 23 лицензии Министерства связи Российской Федерации " 5769 по обеспечению проведения оперативно-розыскных мероприятий. ЗАО "Байярд-Славия Коммуникэйшнс" согласовывает с Управлением ФСБ по Волгоградской области список лиц из числа своих сотрудников, обслуживающих узел передачи данных ЗАО "Байярд-Славия Коммуникэйшнс". которые будут допущены к работе по обеспечению проведения оперативно-розыскных мероприятий на узле предачн данных, принимает меры к недопущению раскрытия приемов ведения указанных мероприятий.

Срок:
отв.: ЗАО "Байярд-Славия Коммуникэйшнс"

2. Управление ФСБ по Волгоградской области предоставляет список лиц из числа сотрудников УФСБ, которые будут со стороны Управления ФСБ по Волгоградской области проводить оперативно-розыскные мероприятия на узле передачи данных ЗАО "Байярд-Славия коммуинкэйшнс".

Срок:
отв.: Управление ФСБ по Волгоградской области

3. ЗАО "Байярд-Славия Коммуникэйшнс" демонстрирует сотрудникам Управления ФСБ по Волгоградской области (согласно п. 2) возможность проведения оперативно-розыскных мероприятий на узле передачи данных ЗАО "Байярд-Славия Коммуникэйшнс".

Срок:
отв.: ЗАО "Байярд-Славия Коммуникэйшнс"

4. Сотрудники Управления ФСБ по Волгоградской области (согласно п. 2) изучают совместно с операторами ЗАО "Байярд-Славия Коммуникэйшнс'' (согласно п. 1) алгоритм проведения оперативно-розыскных мероприятий на узле передачи данных ЗАО "Байярд-Славня Коммуникэйшнс" ЗАО "Байярд-Славия Коммуникэйшнс" рекомендует перечень оборудования, необходимого Управлению ФСБ по Волгоградской области для проведения оперативно-розыскных мероприятий на узле передачи данных ЗАО "Байярд-Славня Коммуиикэйшнс" в режиме удаленного доступа.

Срок:
отв.: ЗАО "Байярд-Славия Коммуникэйшнс"

5. Управление ФСБ по Волгоградской области проводит опытную эксплуатацию аппаратно-программного комплекса мониторинга виртуальных клиентов на базе оборудования оператора узла передачи данных ЗАО "Байярд-Славня Коммуникэйшнс".

Срок:
отв.: ЗАО "Байярд-Славия Коммуникэйшнс"

6. Управление ФСБ по Волгоградской области оценивает степень готовности аппаратно-программного комплекса к проведению оперативно-розыскных мероприятий на узле передачи данных ЗАО "Байярд-Славия Коммуникэйшнс", регистрируется на узле передачи данных ЗАО "Байярд-Славия коммуникэйшнс". получая возможность удаленного доступа к данным, получаемым в результате проведения оперативно-розыскных мероприятий на узле передачи данных ЗАО "Байярд-Славия Коммуникэйшнс".

отв.: Управление ФСБ по Волгоградской области
