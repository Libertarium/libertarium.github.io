ПЛАН
Реализации ЗАО "Байярд-Славия Коммуникэйшнс", предоставляющим услуги передачи данных на территории Волгоградской области, особых условий лицензии Госкомсвязи РФ "5769 по обеспечению оперативно-розыскных мероприятий (ОРМ)

В соответствии с Конституцией РФ, Федеральными законами "Об оперативно-розыскной деятельности", "О связи", "Об информации, информатизации и защите информации"(ст. 11),с Гражданским Кодексом РФ (ч.1,ст,139), Уголовно-процессуальным Кодексом РФ (ст.ст.12,174), Уголовным Кодексом РФ (ст.ст.137,138), а также требованиями лицензии на предоставление услуг телематических служб " 5769, для обеспечения проведения оперативно-розыскных мероприятии Управлением ФСБ РФ по Волгоградской области ЗАО "Байярд-Славия Коммуникэйшнс", обеспечивающее функционирование сетей передачи данных, и УФСБ РФ по Волгоградской области принимают на себя следующие обязательства по внедрению системы технических средств по обеспечению ОРМ на узле ПД, принадлежащем ЗАО "Байярд-Славия Коммуникэйшнс":

1. ЗАО "Байярд-Славия Коммуникэйшнс" согласовывает с УФСБ РФ по Волгоградской области список лиц из числа своих сотрудников, обслуживающих узел ПД ЗАО "Байярд-Славня Коммуникэйшнс", которые будут допущены к работе по обеспечению проведения ОРМ на узле ПД, принимает меры к недопущению раскрытия приемов ведения указанных мероприятий

Срок: 28 апреля 1999г.
Ответств.: ЗАО "Байярд-Славия Коммуникэйшнс"

2. УФСБ РФ по Волгоградской области предоставляет список лиц из числа своих сотрудников, которые будут со стороны УФСБ РФ по Волгоградской области проводить ОРМ на узле ПД ЗАО "Байярд-Славия Коммуникэйшнс"

Срок: 30 апреля 1999 г.
Ответств.: УФСБ РФ по Волгоградской области

3. ЗАО "Байярд-Славия Коммуникэйшнс" демонстрирует сотрудникам УФСБ РФ по Волгоградской области (согласно п.2) возможность проведения ОРМ на узле ПД ЗАО "Байярд-Славия Коммуникэйшнс"

Срок: 5 мая 1999 г.
Ответств. : ЗАО "Байярд-Славия Коммуникэйшнс"

4. Сотрудники УФСБ РФ по Волгоградской области (согласно п.2) изучают совместно с операторами ЗАО "Байярд-Славия Коммуникэйшнс" (согласно п. 1) алгоритм проведения ОРМ на узле ПД ЗАО "Байярд-Славия Коммуникэйшнс".
ЗАО "Байярд-Славия Коммуникэйшнс" рекомендует перечень оборудования, необходимого УФСБ РФ по Волгоградской области для проведения ОРМ на узле ПД ЗАО "Байярд-Славия Коммуникэйшнс" в режиме удаленного доступа.

Срок: 20 мая 1999 г.
Ответств. : ЗАО "Байярд-Славия Коммуникэйшнс"

5. УФСБ РФ по Волгоградской области проводит опытную эксплуатацию аппаратно-программного комплекса мониторинга виртуальных клиентов на базе оборудования оператора узла ПД ЗАО "Байярд-Славия Коммуникэйшнс".

Срок: 20 мая 1999 г.
Ответств. : ЗАО "Байярд-Славия Коммуникэйшнс"

6. УФСБ РФ по Волгоградской области оценивает степень готовности аппаратно-программного комплекса к проведению ОРМ на узле ПД ЗАО "Байярд-Славия Коммуникэйшнс", регистрируется на узле ПД ЗАО "Байярд-Славия Коммуникэйшнс", получая возможность удаленного доступа к данным, получаемым в результате проведения ОРМ на узле ПД ЗАО "Байярд-Славия Коммуникэйшнс".
